# 모바일 점검 확인표

아래 기기/뷰포트에서 최소 1회씩 확인합니다.

- iPhone SE (375x667)
- iPhone 14 (390x844)
- Galaxy S22/S23 유사 규격 (360x780)
- iPad (768x1024)

## 1) 공통 레이아웃

- [ ] 헤더 로고/메뉴/버튼이 겹치지 않는다.
- [ ] 햄버거 메뉴가 열리고 닫힐 때 화면 흔들림이 없다.
- [ ] 본문이 좌우로 잘리지 않고 가로 스크롤이 생기지 않는다.
- [ ] 푸터 링크가 줄바꿈되어도 클릭 영역이 유지된다.

## 2) 메인 페이지

- [ ] 섹션 스냅 이동(휠/터치)이 정상 동작한다.
- [ ] 섹션 카드의 제목/본문이 겹치지 않는다.
- [ ] CTA 버튼(프로그램 보기/상담 신청하기)이 눌리고 잘 보인다.

## 3) 로그인/세션

- [ ] 로그인 폼이 화면 중앙에 표시된다.
- [ ] 로그인 성공 후 운영 대시보드로 이동한다.
- [ ] 로그아웃 버튼(검은색)이 데스크탑/모바일 메뉴에서 동일하게 표시된다.

## 4) 관리자 - 대상자 등록

- [ ] 비밀 유지 서약서 글자가 모바일에서 과도하게 크지 않다.
- [ ] 서명 캔버스가 화면 폭에 맞게 보이고 잘리지 않는다.
- [ ] "서약서 동의 및 다음" 버튼이 하단에서 정상 노출된다.
- [ ] 대상자 정보 입력 폼(date 포함)이 화면을 이탈하지 않는다.

## 5) 관리자 - 목록/필터/페이지네이션

- [ ] 검색조건 입력 버튼으로 연/월 필터를 열고 닫을 수 있다.
- [ ] 연도가 "전체"일 때 월이 "전체"로 고정/비활성화된다.
- [ ] 페이지 크기 선택(10/20/30)이 표 우측 상단에서 동작한다.
- [ ] 목록 탭 전환 시 잔상/깜빡임이 과하지 않다.

## 6) 검사 화면(6도형/인생그래프/성격유형)

- [ ] 툴바 버튼이 모바일에서 줄바꿈되어도 사용 가능하다.
- [ ] 그리기/드래그 지우기 터치 동작이 자연스럽다.
- [ ] 저장 버튼 클릭 시 저장 메시지가 정상 표시된다.
- [ ] "저장된 결과 보기" 링크가 정상 이동한다.
- [ ] 결과 화면에서 "제출 내용 삭제" 버튼이 정상 동작한다.

## 7) 최종 배포 전

- [ ] 콘솔 에러가 없는지 확인한다.
- [ ] 주요 페이지를 1회 이상 새로고침해도 레이아웃 깨짐이 없다.
- [ ] 실제 기기(최소 1대)에서 터치 스크롤/입력/버튼 클릭을 재확인한다.
