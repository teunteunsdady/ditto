# 보류된 정책 결정사항

대화 이력이 사라져도 작업 맥락이 유지되도록, 즉시 적용하지 않은 결정사항을 기록합니다.

## 2026-04-12

### 개인정보 표시 정책 (보류)

- **현재 상태**: 아직 적용하지 않음 (기존 UI 유지)
- **요청 배경**: 상세 화면에서 대상자 이름 노출에 대한 개인정보 보호 이슈 검토
- **결정**: 보호 로직은 준비하되, 사용자가 명시적으로 요청할 때만 적용

### 추후 적용 후보(ON 전환 시 반영)

1. 상세 화면 대상자 이름 기본 마스킹
2. 이름 클릭 시 일시 해제 / 재클릭 시 재마스킹
3. 민감정보 표시 경고 배지 노출
4. 개인정보 조회 이벤트 감사 로그 저장 (Supabase)

### 적용 트리거

- 사용자가 아래와 같은 요청을 보낼 때 적용:
  - "개인정보 마스킹 적용해줘"
  - "보류해둔 보호정책 켜줘"
  - "상세 이름 토글 로직 적용해줘"
